## 🚀 Sobre o Projeto

**EcoScanApp** é uma solução inteligente criada para aproximar tecnologia e consciência ambiental. Utilizando modelos de ponta de **Visão GPT-4o**, o app identifica objetos em tempo real para fornecer métricas ecológicas detalhadas, ajudando usuários a tomar decisões mais conscientes sobre resíduos e consumo.

- 🔍 **Reconhecimento Inteligente** — IA avançada detecta materiais e produtos via câmera ou upload de imagem
- 📊 **Análise Quantitativa** — Calcula a pegada de carbono (CO₂) e estima o tempo de decomposição
- 💡 **Insights Práticos** — Sugere formas criativas de reutilização e informa sobre reciclabilidade
- 🌍 **Eco-Analogias** — Converte dados em estatísticas compreensíveis (ex: equivalente em árvores plantadas)

---

## 📈 Métricas Ambientais

| Métrica | Descrição |
|:---|:---|
| 🌍 **Pegada de Carbono** | Cálculo preciso de CO₂ com base no tipo e quantidade do material |
| 🚦 **Barra de Impacto** | Feedback visual (Baixo / Médio / Alto) para avaliação ambiental imediata |
| 🌳 **Eco-Analogias** | Converte emissões em estatísticas intuitivas — árvores, km de carro, etc. |
| ♻️ **Economia Circular** | Sugestões diretas de reaproveitamento e gestão inteligente de resíduos |

## 📦 Instalação & Execução

**1. Clone o repositório**

```bash
git clone https://github.com/dhiogoFrutuoso/EcoScanApp.git
cd EcoScanApp
```

**2. Configure sua chave da API**

```bash
# Crie um arquivo .env na raiz do projeto
OPENAI_API_KEY=sua_chave_aqui
```

**3. Instale as dependências e execute**

```bash
pip install streamlit openai python-dotenv
streamlit run ecoscan.py
```

Acesse `http://localhost:8501` no navegador e comece a escanear! 🌿
